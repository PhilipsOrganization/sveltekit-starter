# Sveltekit Starter Pack

My personal Sveltekit Setup.

Includes:

- Tailwind CSS
- Auth + Redirects
- Basic Routes
- Action Schema Validation
- Toasts
- Docker Setup + Traefik
- Deploy Script
- Service Worker
- Helpers

## Setup

Copy the `.env.example` file to a `.env` file and update the values.

## Usage

```bash
bun install
bun run dev
```
